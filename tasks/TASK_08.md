## 화면 구성
```
┌─────────────────────────┐
│  발표 완료               │
│  총 시간: 04:32          │
├─────────────────────────┤
│  filler  총 12회         │
│  어 5회 / 음 4회 / 그 3회 │  ← 상위 3개
├─────────────────────────┤
│  속도 이탈               │
│  빠름 3회 / 느림 1회      │
├─────────────────────────┤
│  섹션별 시간              │
│  오프닝    30s / 목표 30s ✓│
│  문제정의  78s / 목표 60s ✗│  ← 초과 시 빨간색
│  솔루션    85s / 목표 90s ✓│
├─────────────────────────┤
│  [다시 발표] [홈으로]     │
└─────────────────────────┘
```
