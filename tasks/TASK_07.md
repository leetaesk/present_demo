# TASK 07 — Prepare 페이지 (설계 페이지)

## 목표
발표 전 설정 입력 화면. 설정 저장 후 `/live`로 이동.

## 만들 파일
- `src/app/prepare/page.tsx`

## 화면 구성
```
┌─────────────────────────┐
│  발표 준비               │
├─────────────────────────┤
│  말하기 속도             │
│  최소 [4] ~ 최대 [6] 음절/초 │
├─────────────────────────┤
│  금지 filler             │
│  [어] [음] [그] [+추가]  │  ← 태그 형태, 클릭으로 토글
├─────────────────────────┤
│  섹션 구성               │
│  + 섹션 추가             │
│  ┌──────────────────┐   │
│  │ 오프닝   30초  키워드│  │  ← 각 섹션: 이름, 시간, 키워드
│  │ 문제정의 60초  키워드│  │
│  └──────────────────┘   │
├─────────────────────────┤
│      [발표 시작 →]       │
└─────────────────────────┘
```

## 동작
- 마운트 시 `loadSettings()`로 저장된 설정 불러오기. 없으면 DEFAULT_SETTINGS.
- 섹션 추가/삭제/순서변경 가능
- [발표 시작] → `saveSettings()` → `router.push('/live')`

## 완료 기준
1. DEFAULT_SETTINGS로 초기값 표시
2. 섹션 추가/삭제 동작
3. [발표 시작] 누르면 설정 저장 + `/live` 이동
